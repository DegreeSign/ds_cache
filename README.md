## Setup
Install using `yarn add @degreesign/cache` or `npm install @degreesign/cache` 

OR use in browsers through CDN

```html
<script 
  src="https://cdn.jsdelivr.net/npm/@degreesign/cache@0.0.1/dist/browser/degreesign.min.js"
></script>
```
